#  AI Meeting Productivity Analyzer

## Project Overview

The **AI Meeting Productivity Analyzer** is a smart dashboard application that analyzes meeting data and provides insights into productivity, duration, and cost.
It helps organizations understand how effective their meetings are and identify areas for improvement.

---

##  Features

*  Interactive dashboard using Streamlit
*  Upload CSV dataset dynamically
*  Productivity analysis and visualization
*  Cost vs Duration insights
*  Data filtering and exploration
*  Fast and user-friendly interface

---

##  Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* Matplotlib & Seaborn

---

##  Dataset Format

The uploaded CSV file should contain the following columns:

* `meeting_day` → (Monday, Tuesday, etc.)
* `meeting_type` → (Standup, Review, Planning, etc.)
* `duration_min` → Duration of meeting in minutes
* `cost_inr` → Total meeting cost in ₹

---

##  How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI_Meeting_Cost.git
cd AI_Meeting_Cost
```

### 2. Install dependencies

```bash
pip install pandas streamlit scikit-learn matplotlib seaborn joblib
```

### 3. Run the app

```bash
streamlit run app.py
```

### 4. Open in browser

```
http://localhost:8501
```
